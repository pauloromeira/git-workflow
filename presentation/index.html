Criação

Criação da Task


  # 1. Criação da branch da Task a partir da master atualizada
  git checkout master
  git pull origin master
  git checkout -b feature/LEGYS-1500 # Cria branch da Task

  # 2. Criação do esqueleto da spider e atualização das settings.py
  git status # Verifica arquivos criados / modificados
  git add . # Adiciona arquivos - "." vai adicionar o diretorio atual inteiro
  git commit -m "[feature/LEGYS-1500] Cria esqueleto da spider <nome-do-tribunal>"
  git push origin feature/LEGYS-1500 # Atualiza o Github
